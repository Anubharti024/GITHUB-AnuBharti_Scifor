HTML Elements

An HTML element is a fundamental building block of a webpage, consisting of a start tag, content (optional), and an end tag, which together define the element's structure and functionality.

  • <h1 > My First Heading.
  • <p > My First Paragraph.

What are Semantic Elements?

A semantic element clearly describes its meaning to both the browser and the developer.

Examples of Semantic elements: <section>,<article>,<nav> etc.

Examples of non-Semantic elements: <div> and <span> - Tells nothing about its content.

Example

<!DOCTYPE html>
<html>
<head>
<title>Second Page</title>
</head>
<body>
<section>
<h1>This is Semantic Element Example. </h1>
<p>A semantic element clearly describes its meaning.</p>
</section>
</body>
</html>
